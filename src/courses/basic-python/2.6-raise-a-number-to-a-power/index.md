## Raise a number to a power

In Python, the operator `**` is used to raise a number to a power. The syntax for this operation is as follows:

```shell
base ** exponent
```

Here's an example:

```python
# raise 2 to the power of 3
result = 2 ** 3
print(result)  # output: 8

# which is the same as
result = 2 * 2 * 2
print(result) # output: 8
```
