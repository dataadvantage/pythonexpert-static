## Multiplying two numbers together

The multiplication operator `*` is used to multiply two numbers. It works with both integers and floating-point numbers.

For example:

```python
a = 3
b = 4
c = a * b
print(c)  # output: 12

x = 2.5
y = 1.5
z = x * y
print(z)  # output: 3.75
```
